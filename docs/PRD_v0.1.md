# 台股持股管理系統 PRD v0.1

## 1. 產品目標

建立一套以台股為主的個人投資管理系統，讓使用者能夠：

1. 用 Yahoo 股市截圖快速匯入持股資訊。
2. 管理台股持股與觀察清單。
3. 即時或準即時更新個股行情。
4. 查看個股技術分析。
5. 設定買進、賣出、停損與技術條件提醒。
6. 透過 Gmail 收到價格與技術提醒。
7. 後續擴充個股新聞、營收、EPS、法人進出、殖利率與 AI 每日健檢。

## 2. 使用者情境

### 情境 A：快速匯入持股

使用者在 Yahoo 股市 App 中截圖目前持股，進入系統上傳截圖。系統 OCR 辨識股票代號、名稱、股數、成本，並以表格呈現。使用者確認後匯入。

### 情境 B：盤中查看損益

使用者打開持股總覽，系統抓取目前行情，顯示總成本、總市值、未實現損益、今日漲跌。

### 情境 C：檢查個股技術面

使用者點選台積電或 00631L，系統顯示 K 線、均線、RSI、MACD、KD、布林通道，並給出技術分數與文字判讀。

### 情境 D：價格提醒

使用者設定台積電跌到 980 元提醒、00631L 跌破 20MA 提醒。系統檢查後若觸發條件，寄 Gmail 通知。

## 3. MVP 功能範圍

### 3.1 持股管理

欄位：

- 股票代號
- 股票名稱
- 市場：上市 TW / 上櫃 TWO
- 股數
- 平均成本
- 備註

### 3.2 Yahoo 截圖匯入

- 上傳圖片
- OCR 辨識
- 手動貼上 OCR 文字
- 解析股票代號、名稱、股數、平均成本
- 匯入前人工確認

### 3.3 行情與損益

- 取得現價
- 計算成本總額
- 計算市值
- 計算未實現損益
- 計算報酬率

### 3.4 技術分析

指標：

- 日 K
- 5MA、10MA、20MA、60MA、120MA、240MA
- RSI14
- MACD 12/26/9
- KD 9
- 布林通道 20/2
- ATR14

判讀：

- 站上 / 跌破 20MA
- 站上 / 跌破 60MA
- 均線多頭排列
- RSI 過熱 / 超賣
- MACD 翻正 / 翻負
- 放量上漲 / 放量下跌

### 3.5 提醒

支援規則：

- price：價格大於、小於、等於門檻
- ma20_cross_down：跌破 20MA
- ma20_cross_up：站上 20MA
- rsi：RSI 大於或小於門檻

通知方式：

- Gmail SMTP

## 4. 非 MVP 功能

以下排入 v0.2 之後：

- 個股新聞與 AI 摘要
- 交易紀錄與已實現損益
- 年度股利統計
- ETF 專頁
- Telegram Bot 通知
- 使用者登入與雲端同步
- 手機 PWA 安裝

## 5. 風險與限制

1. Yahoo 股市截圖格式可能因版本、手機解析度不同而影響辨識。
2. yfinance 台股資料可能延遲、缺漏或盤中資料不穩。
3. Gmail SMTP 需使用應用程式密碼。
4. 技術指標僅為輔助資訊，不構成投資建議。
5. 若要商業化提供他人使用，行情資料授權需另行處理。
