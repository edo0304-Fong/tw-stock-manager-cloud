# 開發路線圖

## v0.1：本機 MVP

- Streamlit Web App
- Yahoo 股市截圖 / OCR 文字匯入
- 持股總覽
- yfinance 行情更新
- 個股技術分析
- Gmail 價格與技術提醒

## v0.2：台股資料完整化

- 接 FinMind：月營收、EPS、股利、法人買賣超
- 接 TWSE / TPEx OpenAPI：上市櫃基本資料、官方資料補強
- 每日收盤後自動更新
- 每日持股健檢加入基本面與籌碼面

## v0.3：提醒與報表強化

- Telegram Bot 通知
- 交易紀錄輸入
- 已實現損益
- 年度股利試算
- ETF 分析頁
- 觀察清單與進出場策略

## v0.4：正式 Web App

- Next.js 前端
- FastAPI 後端
- PostgreSQL / Supabase
- 使用者登入
- 雲端部署
- PWA 手機安裝

## v1.0：正式投資中控台

- 合法即時行情 API
- 券商庫存同步
- AI 新聞摘要
- AI 財報摘要
- 風險提醒
- 個人化交易紀律追蹤
